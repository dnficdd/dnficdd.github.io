# Gitee介绍 

Gitee 也是当时和Github 一样，可以正常的立即部署的。

不过，现在新加了一些出了静态页面之外的其他容器之后，就不能免费进行自动部署了，不过，已经有了Github的自动部署，Gitee 就作为一个备份吧。

当然，如果有需要，开通一个会员也是不错的选择，如果有活动赠送会员的话，欢迎补充。

# 腾讯专栏介绍 

腾讯专栏蛮不错的，之前我的文章主要是在cnblog和github，不过后来腾讯有一个活动，就申请了一下，反正文章多重备份无妨。
当时的文章还在咯：
[申请参加腾讯云自媒体分享计划，送代金券+CN域名 ](https://wangkun19930608.github.io/%E6%8A%80%E6%9C%AF/%E7%A0%B4%E8%A7%A3/2018/05/21/blog-trans/ )

并且的话，当时还是不需要手动在这个平台发布的，直接在原来申请的平台发布的文章，会自动同步到腾讯专栏里面去，现在这个功能还有，还是蛮不错的。

不过最近，似乎因为平台出现了其他大量的水文章，现在在平台发布，会进入审核，并不会直接发布，自己的一些笔记什么的，还是放在github吧，这个平台有不错的文章再做备份了。

平台有一个特殊的功能是把已经写好的文章，转换为markdown 的格式，或者反之也是可以的，这个功能很不错，尤其是你的其他平台的图片，都是会自动同步的，不过吗，添加审核之后，易用性感觉大大降低了。
