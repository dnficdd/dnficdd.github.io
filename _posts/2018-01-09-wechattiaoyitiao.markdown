# 实用技巧
跳至盒子中间，+2 分

连续跳至盒子中间，会在上次的基础上累加，+2分

跳至音乐盒上停留会播放音乐，+30分

跳至便利店停留会开门，+15分

跳至魔方停留魔方会旋转，+10分

跳至井盖上停留会有冲水声音，+5分

# 如何解封
之前使用辅助刷的分数，可能被官方屏蔽。如何解封？

第一步： 删除跳一跳游戏
第二步：退出微信账户
第三步：重新登陆微信下载跳一跳游戏
然后等待 ，被锁的跳一跳游戏分数就会清0并解封！
